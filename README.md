# Decision Tree Tutorial: Effect of Tree Depth on Overfitting
## Overview

This project explores how the depth of a Decision Tree affects model performance, specifically focusing on underfitting and overfitting.
The tutorial demonstrates how increasing tree depth impacts training and testing accuracy, and highlights the importance of selecting an appropriate model complexity for better generalisation.

---
## Objectives
- Understand how Decision Trees work
- Learn the concept of tree depth
- Explore underfitting and overfitting
- Analyse model performance using training and testing accuracy
- Identify optimal tree depth for best generalisation
---
## Dataset
The project uses the Breast Cancer Wisconsin Diagnostic Dataset
- Binary classification problem:
  - Benign
  - Malignant
- Loaded directly using `sklearn`
---
## How to Run
1. Open the notebook.
2. Run all cells from top to bottom.
3. The notebook will:
- Load the dataset
- Train models with different tree depths
- Generate plots and visualisations
No external dataset download is required.

---
## Key Results
- Increasing tree depth improves training accuracy
- Test accuracy improves only up to a certain depth
- Deeper trees can lead to overfitting
- Optimal depth balances performance and generalisation
